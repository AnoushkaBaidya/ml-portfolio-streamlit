# ML Portfolio Streamlit App

A multi-page machine learning portfolio application built with Streamlit.

## Included Projects

- Diabetes Prediction
- Netflix Clustering
- Spotify Popularity Prediction
- Customer Churn Prediction

## Goals

- Present multiple ML projects through one recruiter-friendly live link
- Show classification, regression, clustering, explainability, and threshold tuning
- Use professional project structure and modular Python code

## Run locally

```bash
streamlit run app.py
